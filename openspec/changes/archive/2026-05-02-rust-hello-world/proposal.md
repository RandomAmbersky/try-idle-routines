## Why

В репозитории ещё нет минимального Cargo-проекта; нужна стандартная заготовка «Hello, world!», чтобы зафиксировать структуру, команды сборки и основу для дальнейшей idle RTS-логики на Rust.

## What Changes

- Добавить в корень репозитория стандартный **Cargo-проект** (бинарный крейт) с `main`, печатающим приветствие.
- Зафиксировать ожидаемое поведение и команды проверки в спеках и задачах реализации (без изменения игрового дизайна).

## Capabilities

### New Capabilities

- `rust-bootstrap`: минимальный собираемый Rust-бинарник в корне репо, вывод приветствия в stdout, ожидаемые команды `cargo build` / `cargo run` / `cargo test` (если добавлен smoke-тест).

### Modified Capabilities

- *(нет — существующих спеков в `openspec/specs/` пока нет.)*

## Impact

- Корень репозитория: появятся `Cargo.toml`, `src/main.rs`, при необходимости `Cargo.lock` и минимальный тест или только документированная проверка руками.
- Зависимости: только стандартная библиотека Rust (без внешних крейтов для hello world).
- CI/инструменты: при появлении пайплайна — шаг `cargo build` (вне scope этого изменения, если CI ещё нет).
