## ADDED Requirements

### Requirement: Корневой Cargo-проект собирается без внешних зависимостей

Репозиторий SHALL содержать в корне манифест `Cargo.toml`, описывающий один исполняемый пакет (binary crate), и исходник точки входа так, что зависимости ограничены стандартной библиотекой Rust для реализации приветствия.

#### Scenario: Успешная сборка

- **WHEN** в корне репозитория выполняется команда `cargo build`
- **THEN** сборка завершается без ошибки

### Requirement: Бинарь выводит приветствие в стандартный вывод

При запуске собранного бинаря из корня репозитория SHALL в stdout появиться строка, содержащая текст `Hello, world!` (после чего процесс завершается успешно).

#### Scenario: Запуск через Cargo

- **WHEN** в корне репозитория выполняется команда `cargo run`
- **THEN** в stdout содержится подстрока `Hello, world!` и код выхода процесса нулевой
