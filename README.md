# idle-tui

Terminal **idle game** (Rust + [ratatui](https://github.com/ratatui-org/ratatui)) with a UFO / X-COM–style containment fantasy: incidents, squads, rising pressure.

Unlike classics in that lineage, **there is no strategic command layer** you run by hand — **missions surface on the map**, and **squads pick suitable missions, complete them, return to base**, and repeat as time advances.
